Informações sobre a Chave DataLattes
A Chave DataLattes é um arquivo(.Dtkey) gerado pelo sistema de forma a armazenar a descrição de critérios selecionados. Essa chave é utilizada para carregar os critérios de análise e suas pontuações antes de submeter o currículo Lattes no sistema.
A Chave é gerado ao finalizar o processo de especificação dos critérios. Para especificar os critérios selecione a opção “Criar Chave” na página inicial. Ou se preferir clique aqui
Durante alguma atualização do sistema Datalattes a Chave criada anteriormente pode apresentar defasagem de versão, ou seja, ela foi criada para uma versão anterior do sistema. Para solucionar esse problema é necessário especificar uma nova Chave, clique aqui para criar uma nova
