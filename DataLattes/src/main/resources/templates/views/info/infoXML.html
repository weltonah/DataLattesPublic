Informações sobre o arquivo XML
Um arquivo XML, é um documento utilizado para armazenar e para transportar dados de forma organizada. Nesse sistema são utilizados arquivos em formato XML da base de dados da Plataforma Lattes.
A Plataforma Lattes estabelece um padrão de currículo que é largamente utilizado principalmente no meio acadêmico brasileiro. Além de estabelecer o padrão ele também armazena e disponibiliza de forma pública esses dados. Geralmente o acesso ocorre através de buscas pela própria Plataforma Lattes. Os dados de cada usuário cadastrados podem ser visualizados e também podem ser baixados. A forma que a Plataforma Lattes estabeleceu para disponibilizar esses dados para serem baixados é através de um arquivo XML. Cada arquivo XML baixado corresponde aos dados vinculados a um Usuário buscado na Plataforma Lattes.
Ao acessar a Plataforma Lattes é possível buscar por currículos através dos nomes dos usuários cadastrados. Clique aqui para acessar a página de busca. Ao selecionar um currículo, será mostrado todos os dados cadastrados pelo o usuário, constando local de formação, linhas de pesquisas, artigos escritos, participação de bancas entre outros itens. Para baixar o arquivo do respectivo currículo, é necessário clicar no ícone “XML” que se encontra na parte superior da página como na figura abaixo.
