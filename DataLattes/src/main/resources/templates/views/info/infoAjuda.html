O sistema é dividido em 2 duas partes. A primeira parte é a responsável pela criação da Chave. Durante o processo é necessário selecionar os anos de validade, os critérios e seus respectivos valores e limites de contabilização se existir. Ao finalizar o processo o sistema vai disponibilizar o arquivo de Chave. Na segunda parte do sistema é necessário utilizar a Chave gerada para submeter os critérios, e logo após surge um formulário para submeter o currículo Lattes em formato XML que vai ser analisado. Após a análise é mostrado a pontuação final e o detalhamento de dos itens analisados. Se for necessário é possível importar os dados do resultado em formato compatível com Excel.
